API

Convert a PDF's first page to JPEG or PNG over plain HTTP — from n8n, curl, or any tool that can make a POST request.

Status
Live
Formats
JPEG, PNG
Max file size
20 MB
Pages converted
First page only

Request

curl -X POST https://pdfconverter-nu.vercel.app/api/convert \ -F "file=@yourfile.pdf" \ -F "format=jpeg" \ -o data

Parameters

FieldTypeRequiredDescription
file form-data file Yes The PDF to convert.
format form-data text No Output format: jpeg (default) or png.

Response

On success, returns the rendered image directly as the response body (Content-Type: image/jpeg or image/png). On error, returns a JSON body with an error field and a 4xx/5xx status code.
